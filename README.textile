h1. Comitiva - Gerência de Eventos

  *Comitiva* é um sistema para gerência de eventos desenvolvido sob o framework "CakePHP":http://cakephp.org


h2. Documentação

h3. Requisitos Mínimos

Requistos mínimos:

* CakePHP 2.x
* MySQL 5.x ( é possível utilizar outros SGDB com pouca ou nenhuma alteração de código )
* PHP >= 5.3

Contamos com a colaboração de todos. O projeto é voluntário e nem sempre temos tempo para documentar. Algumas fontes de documentação:

* "Como utilizar controle de permissão no sistema":http://cauancabral.net/2010/08/01/comitiva-como-utilizar-controle-de-permissao-no-sistema
* "Como utilizar controle de permissão no sistema – quase tudo mudou":http://cauancabral.net/2010/08/09/comitiva-como-utilizar-controle-de-permissao-no-sistema-quase-tudo-mudou
